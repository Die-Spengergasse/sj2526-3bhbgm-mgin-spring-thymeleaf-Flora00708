// wie eine foreach schleife, p ist das aktuelle element, patients ist die liste // ${p.name} gibt den Namen des aktuellen Patienten aus, der in der Variable p gespeichert ist. Das th:text-Attribut sorgt dafür, dass der Textinhalt der Zelle mit dem Wert von p.name ersetzt wird. // ${p.xxx} wird übergeben von der Controller-Methode, die die Patientenliste bereitstellt. In diesem Fall wird die Liste der Patienten wahrscheinlich als Modellattribut mit dem Namen "patients" an die View übergeben, und jedes Element in dieser Liste hat eine Eigenschaft "name" und "birth", die hier angezeigt werden.
